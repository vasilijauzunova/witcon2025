Instagram Embed Block Test

This page demonstrates the Instagram embed block with different usage patterns.

Auto-Blocked Pattern (Simple URL)

The simplest way to embed an Instagram post - just paste the URL:

https://www.instagram.com/p/C1234567890/

Auto-Blocked Pattern (With Preview Image)

You can also add a preview image before the URL for better loading experience:

https://www.instagram.com/p/C9876543210/

Explicit Embed Block

For more control, you can use an explicit Embed block:

Embed
https://www.instagram.com/p/CxYz123456/

Instagram Reel

The embed block also works with Instagram Reels:

https://www.instagram.com/reel/D1234567890/
